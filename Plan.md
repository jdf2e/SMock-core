### 整个项目流程

    1. 开始 -> 
    2. 根据文档类型进入不同功能分支 -> 
    3. 获取对应的假数据文件（接口或者json文件性质） -> 
    4. 把所有数据全部转换成对应的SMockData类型 ->
    5. 基于SMockData生成本地服务的数据文件（可以合并为一个文件也可以单独存在） ->
    6. 基于SMockData生成URL的Map ->
    7. 基于URLMap创建服务接口 ->
    8. 启动服务以供用户访问 ->
    9. 访问接口是校验入参是否合法
